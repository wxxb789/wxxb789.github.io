#+BEGIN_QUOTE
简单来说，就是不使用 /continue 命令，而是使用 /start 命令+小节名，让 ChatGPT 从给定的小节开始，个人认为很适合在网页版中用。
由于 OpenAI 的 API 目前仍不支持持续对话，tokens 的消耗对于这个 prompt 来说确实有点高，期待继续优化
另外也可以看看 @blackanger 大佬在尝试的一些有意思的优化
